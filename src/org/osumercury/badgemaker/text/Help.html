Help Topics

Mercury Badge Maker (mercbm) is a software that can be used to generate badges and certificates. The following is the user workflow for the program:

  1. Enter individuals' data using the interface in the Input tab
  2. Manipulate data as needed
  3. Specify format for the badge or certificate using the Format tab
  4. Generate outputs (PDF or PNG / JPG images) using the Output tab

  3. Create Badges
  4. Create Certificates
